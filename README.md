# mnist-pca-dimensionality-reduction
Dimensionality reduction of MNIST dataset using PCA in Python
# Анализ изображений рукописных цифр MNIST с использованием PCA

Учебный проект по машинному обучению и линейной алгебре.  
В проекте рассматривается применение метода главных компонент — PCA — для анализа и снижения размерности изображений рукописных цифр из датасета MNIST.

## Описание проекта

Каждое изображение MNIST имеет размер 28×28 пикселей.  
После преобразования в вектор одно изображение описывается 784 признаками.

Цель проекта — показать, как с помощью PCA можно уменьшить количество признаков, сохранив основную информацию о форме цифры.

## Что делает программа

Скрипт выполняет следующие шаги:

- загружает датасет MNIST через `sklearn.datasets.fetch_openml`;
- нормализует значения пикселей к диапазону `[0, 1]`;
- строит PCA-проекцию данных на 2 компоненты для визуализации;
- рассчитывает количество главных компонент для сохранения 90%, 95% и 99% дисперсии;
- показывает восстановление изображений при разном количестве компонент;
- сохраняет графики в папку `figures/`.

## Установка и запуск

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить проект:

```bash
python mnist_pca_project.py
```

Важно: при первом запуске требуется интернет, так как датасет MNIST загружается через OpenML.  
После загрузки данные кешируются локально.

## Результаты

В ходе проекта показано, что PCA позволяет значительно уменьшить размерность изображений MNIST.

Исходно каждое изображение описывается 784 признаками.  
После применения PCA можно оставить меньшее число главных компонент и при этом сохранить основную структуру изображения.

Проект демонстрирует:

- визуализацию многомерных данных в двумерном пространстве;
- анализ накопленной объяснённой дисперсии;
- восстановление изображений после сжатия;
- практическое применение линейной алгебры в задачах анализа данных.
